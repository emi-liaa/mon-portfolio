Application web "Gestion de Tâches"

Objectifs du Projet

Concevoir et structurer une interface utilisateur complète pour une application de gestion de tâches (To-Do List). L'accent a été mis sur la sémantique HTML et la qualité du stylage CSS pour garantir une bonne expérience utilisateur dès les bases.

Compétences Démontrées

  • Structure: Utilisation de balises sémantiques HTML5 (sections, articles, listes) pour une application claire.
  • Style: Maîtrise des propriétés CSS de base (couleurs, fonds, polices) pour une interface agréable.
  • Organisation: Capacité à organiser et présenter clairement des informations complexes (liste de tâches).
  • Design: Création d'une interface simple et intuitive pour faciliter la saisie et la gestion des tâches.
